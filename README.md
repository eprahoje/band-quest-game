# band-quest-game

Implementação do jogo Band Quest — RPG de gerenciamento de banda de rock,
para navegador e Android.

## Documentação

A especificação completa está em `../band-quest-docs`. Antes de implementar
qualquer feature, leia:

1. `../band-quest-docs/docs/memory-bank/product.md` — visão de produto e decisão de MVP visual.
2. `../band-quest-docs/docs/memory-bank/architecture.md` — decisões de arquitetura cross-feature.
3. A pasta da feature em `../band-quest-docs/docs/features/<slug>/`:
   - `planning/overview.md`
   - `refinement/iteration-XX.md` (mais recente)
   - `refinement/decomposition.md`
